# Spotify Customer Churn Prediction

## 📌 Project Overview

Customer retention is one of the biggest challenges for subscription-based businesses. Predicting customers likely to discontinue their subscriptions enables companies to take proactive retention measures.

This project develops an Artificial Neural Network (ANN) to predict customer churn using customer behavior and subscription-related features.

---

## 🎯 Objectives

- Analyze customer behavior
- Predict customer churn
- Identify key churn indicators
- Evaluate ANN model performance
- Visualize prediction results

---

## 📂 Dataset

Spotify Customer Churn Dataset

Contains customer demographics, subscription details, listening behavior, and churn status.
